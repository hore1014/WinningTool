# WinningTool

A Tool to automatically generate inputs for the IBSYS 2 Supply Chain Simulator

## Einrichten von SQLite
Die vorkompilierten Binaries von SQLite befinden sich im Ordner `.\sqlite\`. Darin befindet sich das Programm `sqlite3.exe` zur ausführung von SQLite. Um diese im Programm oder in der Kommandozeile auszuführen, muss das oben genannte Verzeichnis des Programms zur PATH Umgebungsvariable hinzugefügt werden.
